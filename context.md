# Project Context

## Goal
Build a polished 30-second Remotion video with modern motion graphics, kinetic typography, and smooth transitions that ranks the top 5 vibe coding tools (Vibecode, Rork, Replit, Cursor, Rocket) for mobile development.

## Audience
Beginners who are just starting with vibe coding and want to ship their first mobile app.

## Video Specs
- **Duration**: 30 seconds
- **Resolution**: 1920x1080
- **Frame rate**: 30fps
- **Total frames**: 900

## Video Structure
| Section | Time | Frames | Description |
|---------|------|--------|-------------|
| Intro | 0-4s | 0-120 | Title animation with blur reveal |
| Tool #1 - Vibecode | 4-8.4s | 120-252 | Green accent, "Mobile-First Winner" |
| Tool #2 - Rork | 8.4-12.8s | 252-384 | Blue accent, "Own Your Code" |
| Tool #3 - Replit | 12.8-17.2s | 384-516 | Orange accent, "Cloud Playground" |
| Tool #4 - Cursor | 17.2-21.6s | 516-648 | Purple accent, "Pro's Workbench" |
| Tool #5 - Rocket | 21.6-26s | 648-780 | Red accent, "One to Watch" |
| Outro | 26-30s | 780-900 | Summary cards + CTA |

## Tech Stack
- Remotion (React-based video framework)
- TypeScript
- React
